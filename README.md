# QuantumAI Solutions — Enterprise AI Engineering

> We design and deploy production-grade AI agents, RAG systems, Voice AI Calling Agents, and Workflow Automation Platforms for modern businesses.

## 🌐 Live Website
[quantumai.in](https://quantumai.in)

## 🛠️ Tech Stack
- **Frontend**: HTML, CSS, Vanilla JS (Static)
- **Backend**: Node.js + Express.js
- **Database**: MongoDB Atlas (Mongoose ODM)
- **Deployment**: Vercel

## 📦 Features
- Production-grade AI Agent showcase
- RAG-powered Knowledge Base demos
- Voice AI Calling Agent
- ContentForge AI — multi-platform content publishing
- Lead capture forms → MongoDB Atlas
- Book-a-Demo scheduling

## 🚀 Local Development

### Backend
```bash
cd backend
npm install
cp .env.example .env  # Fill in your MongoDB URI
npm start
```

### Frontend
```bash
npx http-server website -p 8081
```

## ⚙️ Environment Variables
See `backend/.env.example` for required variables.
